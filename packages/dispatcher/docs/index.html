How it works

Three steps. Zero friction.

No IDE required. No SSH tunnels. Just your phone and your AI agent.

💬
Step 1

Send a message

Describe your task in Telegram, just like texting a colleague. Mention a project name or let Dispatcher auto-detect it.

⚙️
Step 2

Agent executes

Dispatcher routes your message to the right project directory and spawns your AI coding agent with full context.

✅
Step 3

Get results

The agent's output is delivered back to Telegram. See diffs, logs, and summaries right on your phone.

Features

Everything you need to code remotely.

Built for developers who want to stay productive away from their desk.

🔄

Session management

Resume conversations by replying to any previous message. Dispatcher keeps full context across sessions.

📂

Project routing

Auto-detect which project you mean from keywords in your message. No need to specify paths manually.

🔗

Smart follow-ups

Short replies like "also fix the tests" auto-link to your most recent task. Natural conversation flow.

⚡

Concurrent tasks

Run multiple AI agents in parallel across different projects. No waiting, no bottlenecks.

🤖

Agent-agnostic

Works with Claude Code, Aider, Cursor, or any CLI-based agent. Swap agents without changing your workflow.

📊

Progress tracking

See typing indicators while agents work. Get periodic status updates on long-running tasks.

🧠

Memory persistence

Dispatcher remembers your preferences, project associations, and patterns across sessions.

🚀

Easy deployment

Install with pip, run one command. No Docker, no infrastructure. Works on any machine with Python 3.10+.

Quick Start

Up and running in 2 minutes.

Four commands. That's all it takes.

1

Install Dispatcher

Requires Python 3.10 or higher.

terminal
pip install agent-dispatcher
2

Initialize configuration

Creates a config file at ~/.config/dispatcher/config.yaml.

terminal
dispatcher init
3

Configure your projects

Add your Telegram bot token, chat ID, and project paths.

~/.config/dispatcher/config.yaml
# Edit with your favorite editor
vim ~/.config/dispatcher/config.yaml
4

Start Dispatcher

That's it. Open Telegram and start sending tasks.

terminal
dispatcher start
Configuration

Simple, declarative config.

One YAML file controls everything. Define your Telegram credentials, choose your AI agent, and register your projects with keyword-based routing.

  • Set your Telegram bot token and authorized chat ID for secure access
  • Choose any CLI agent — Claude Code, Aider, or a custom script
  • Control concurrency limits per machine
  • Register projects with paths and keyword triggers
  • Override agent settings per project when needed
config.yaml
telegram:
  bot_token: "YOUR_BOT_TOKEN"
  chat_id: YOUR_CHAT_ID

agent:
  command: "claude"
  args: ["-p", "--dangerously-skip-permissions"]
  max_concurrent: 3

projects:
  myapp:
    path: ~/projects/myapp
    keywords: [myapp, app]
  backend:
    path: ~/projects/backend-api
    keywords: [backend, api, server]
  docs:
    path: ~/projects/documentation
    keywords: [docs, documentation]
